Silly Goobers generator

By Justin Kemp

A simple, product/webiste that gives a silly animal images to a large variety of uses. Users will be able to input what they need the image for and a pre selected image will be given to them to help solve the issue.

Technology

It will function primarly with Html, Css and Java, and it would function on Chrome, Firefox and most platforms. It will also be hosted on github as well.

Features

It will have a question section to submit your need for the images, a generator and libary for the stored images. You will also be able to save images generated.

Possible Future Features

It will possibly have a customer service section and would have the ability to generate differnt types of images under differnt requirements.

Basics Features

It will be well design and have a simple, yet appealing desgin impleted in the generation proccess. There will be differing types of questions along with hidden images and links in the text of the website.

Advanced

The website will possibly include a rewards points system for using the generator along with the ability to store images in a private user libary. It may also include rare images, that have a lower chance of being generated.
